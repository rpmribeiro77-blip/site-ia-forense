# Checklist de saneamento para organização estratégica do caso

## 1) Objetivo
Organizar o processo para reduzir risco de omissão probatória e retrabalho.

## 2) Itens de verificação
1. Quais são os pontos controvertidos definidos?
2. Quais provas são essenciais para cada ponto?
3. Há pedido de prova técnica? Está bem delimitado?
4. Existe questão processual pendente que pode impactar mérito?
5. O cronograma processual está claro para a equipe?

## 3) Aplicação prática
Use este checklist antes da audiência e antes do encerramento de instrução.

## 4) Ganho real
- Mais previsibilidade processual.
- Melhor organização de prova.
- Redução de falhas de estratégia.
